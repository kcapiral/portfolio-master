A poster design based on typography, color and shapes. The theme of the design was to announce a type talk that would be done by Christian Schwartz from the Commercial Type foundry. The poster uses the font, Graphik, which the mandate was to use a font from the foundry. It was originally produced as a poster project, but the design evolved and adapted to a fictional conference event and displayed on a building billboard, and as a hanging flag design.
