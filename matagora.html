matagora

Type
User interface design
Software
XD, Zeplin

A website designed in collaboration with user experience and researcher, Vithu Namasivayam. My role as the user interface designer involved the following: participation in the creation and reviewing of the sitemap, creating and establishing the brand identity, designing mockups, and making improvements based on changing needs.

As a platform that provides a service to brand owners and space owners, Matagora was built in the framework, Angular, for its fast loading time and easy to implement components. The user interface had to correspond to existing components from the library of Angular Material while presenting as a clean, crafty, and not-so-techy website. The design had to be minimal, to allow for a fluid and smooth experience of the website for scenarios of onboarding users to the platform, searching through space informations, and filling in forms.
