A series of webpages exploring different animations and interactions using HTML, CSS and Javascript. Playground consists of 4 webpages: Less is, Scroll, Tug-o-War and Unveil. “Less is” experiments with simple Jquery transitions, color animations and hover overs. “Scroll” uses CSS animations and Rellax for the movement of typography on scroll. “Tug-o-War” allows for the user to play a game by using the .keypress function. “Unveil” uses Blast.js to select specific words or sentences animates them.
