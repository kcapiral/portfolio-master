playground
2018

Type
Interaction design and Web development
Software
XD, Atom, Github

A series of webpages exploring different animations and interactions using Javascript and libraries. Playground consists of 5 webpages: Less is, Scroll, Ring Ring Ring, Tug-o-War and Unveil. “Less is” experiments with simple Jquery transitions, color animations and hover overs. “Scroll” uses CSS animations and Rellax for the movement of typography on scroll. “Ring Ring Ring” uses setTimeout and setInterval functions to count down the animations of the text. “Tug-o-War” allows for the user to play a game by using the .keypress function. “Unveil” uses Blast.js to select specific words or sentences and to become animated.
